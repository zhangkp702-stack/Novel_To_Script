## 改编任务

- 在现有 YAML 剧本基础上按用户修改要求调整
- 保留未提及部分的核心情节、场景结构与人物关系，只改用户要求涉及的内容
- 遵守下方人物设定，不得擅自改变已定义人物的核心性格
- 旁白、动作、台词仍写入 `场景列表[].剧本正文`，保留 `旁白：` / `动作：` / `人物名：` 等前缀

## YAML 输出规范（最终交付格式）

改编完成后，**只输出修订后的完整中文 YAML**，不要输出解释、分析、前言后记或自然语言剧本正文。

输出要求：

- 从 `文档类型: 按章剧本片段` 开始
- 所有字段名与枚举值均使用中文
- **必须保留 YAML 缩进空格**（子字段相对父字段缩进 2 个空格，与样例一致）
- 严格遵循下方 YAML 样例的结构与字段名
- 可用 ` ```yaml ` 围栏包裹，但不要输出围栏外的任何文字
- 禁止输出 Markdown、JSON 说明、表格
